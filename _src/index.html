Speaking

I returned to Frontend Horse for the Halloween Spooktacular, where Alex and I made a Halloween theme for the Frontend Horse website in 15 minutes using only design tokens.

On Some Antics, Ben Meyers and I talked about using Hacktober as an opportunity to get into design systems while building a small system ourselves.
